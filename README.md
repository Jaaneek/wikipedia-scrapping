# Wikipedia-scrapping

Wikipedia scrapping, made with Chakra-ui, Typescript

## Demo

Project is available at [Link](https://wikipedia-scrapping.vercel.app/)

## Start

```
yarn
yarn start
```

```
npm i
npm run start
```
